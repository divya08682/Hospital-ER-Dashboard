# Hospital Emergency Room Dashboard

## Problem Statement

Hospitals generate massive emergency room data daily. 
Tracking patient wait times, admission status, treatment duration, department workload, and billing manually becomes difficult.

This dashboard helps hospital management:

- Monitor ER patient flow
- Analyze wait times
- Track department performance
- Identify critical cases
- Improve operational efficiency
- Make data-driven decisions

The dashboard converts raw hospital data into actionable insights using Power BI and DAX.
