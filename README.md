# 🏥 Hospital Emergency Room Dashboard

## 📌 Project Overview

The Hospital ER Dashboard is a Power BI project designed to analyze emergency room operations and transform raw hospital data into actionable insights.

This dashboard helps hospital management monitor:

- Patient flow
- Average wait time
- Treatment duration
- Department performance
- Revenue analysis
- Severity level analysis

The project uses CSV datasets, data modeling, DAX measures, and interactive visualizations.

---

## 🚀 Features

- KPI Cards
- Revenue Tracking
- Wait Time Analysis
- Department-wise Analysis
- Severity Distribution
- Interactive Dashboard
- Data Relationships
- DAX Calculations
